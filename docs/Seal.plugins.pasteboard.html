docs » Seal.plugins.pasteboard

Clipboard history backed by the clipboard-cli sync daemon.

Per-keystroke list responses carry only a SUBSTR preview of each entry's text; full text is fetched on paste. All socket traffic goes through nc -U via io.popen — no login shell, no clipboard-cli process startup, no hs.execute round-trip.
